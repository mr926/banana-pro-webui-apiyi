# BananaLab Android

原生 Android 版 BananaLab，基于 Jetpack Compose。

## 打开方式

1. 用 Android Studio 打开 `android/BananaLab`
2. 等待 Gradle Sync 完成
3. 运行 `app`

## 主要能力

- 自定义服务器地址
- 与服务端同密码登录
- 相册选择基础图和参考图
- 图生图生成
- 历史记录和历史相册
- 提示词库管理
- 提示词优化人格管理
- 结果图和历史图下载
- 历史图发送回基础图或参考图

## 备注

当前工程默认使用 `http://127.0.0.1:8787` 作为服务器地址。若你的后端不在本机，请先在设置页里改成实际地址。
